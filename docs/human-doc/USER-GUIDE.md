# USER-GUIDE（Human Readable）

最后更新：2026-03-19

## 如何使用

1. 启动服务：`./target/release/echopup start`（首次可先执行 `cargo build --release`）。
2. 需要管理配置时：`./target/release/echopup ui`，或通过 macOS 状态栏菜单执行同等操作。
3. 在目标应用中按住热键说话，松开后自动输入文本。

## 常见问题

- Q: 热键设置后不生效？
- A: 检查是否使用了受限组合（例如仅普通字母键），并确认已保存配置。

- Q: 模型下载失败或中断？
- A: 重新触发下载，系统支持断点续传与自动重试；可查看日志定位网络问题。

- Q: 后台运行时看不到状态变化？
- A: 在 macOS 上确认状态栏图标存在，且通知与辅助功能权限已授权。
