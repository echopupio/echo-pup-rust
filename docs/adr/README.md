## 约定

1. 每条 ADR 只记录一个决策。
2. 使用递增编号：`0001`、`0002`……
3. 明确状态：拟议 / 已采纳 / 已替代 / 已拒绝。
4. 被替代的 ADR 需要链接到替代记录。
